Requirement  
PHP >= 8.0  
Laravel >= 8.0  
MYSQL >= 5.5

1. Clone Repo
2. Buat DB Baru (karirpad)
3. Import DB
4. cp .env.example .env
5. composer install
6. php artisan migrate
7. npm install  
8. npm run dev
9. php artisan serve
