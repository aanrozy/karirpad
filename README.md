Requirement  
PHP >= 8.0  
Laravel >= 8.0  
MYSQL >= 5.5

1. Clone Repo
2. Buat DB Baru (karirpad)
3. Import DB (karirpad.sql)
4. cp .env.example .env
5. composer install
6. php artisan key:generate
7. php artisan migrate
8. npm install  
9. npm run dev
10. php artisan serve
